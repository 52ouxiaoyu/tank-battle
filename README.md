# 坦克大战 (Tank Battle)

## 👉 [点击开始游戏](https://52ouxiaoyu.github.io/tank-battle/)

经典坦克大战网页版，支持双人对战。

## 操作说明

| 玩家 | 移动 | 射击 | 暂停 |
|------|------|------|------|
| **P1** | W A S D | 空格 | P |
| **P2** | 方向键 | 回车 | P |

## 道具

| 道具 | 效果 |
|------|------|
| ❤️ | 生命 +1 |
| ⭐ | 等级提升 |
| 💣 | 全屏炸弹 |
| 🛡️ | 无敌护盾 |
| 🏗️ | 基地加固 |

## 游戏特色

- 1000 关随机生成，永不重复
- 双人合作模式
- AI 自动托管（5 秒无操作自动接管）
- 连杀自动升级（3/6/10 连杀）
- 多种道具系统
- Boss 战 + 天气特效
- 基地防护系统（5 点血量）
- 冰面滑动效果
- 暂停功能（P 键）
- 最高分记录

## 本地运行

双击 `index.html` 即可。

## 技术栈

纯 HTML + CSS + JavaScript，无需任何依赖。
