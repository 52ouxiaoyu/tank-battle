# 坦克大战：高清重制加强版 (Tank Battle: NES HD Remaster)

这是一个基于经典红白机（NES）《坦克大战》深度重制和增强的网页版游戏。它不仅还原了经典的坦克巷战玩法，还加入了现代化的视觉效果、双人共享生命系统、动态气象 Boss 以及进阶的火力升级机制。

## 🚀 核心特色

### 1. 视觉与显示
*   **极致全屏覆盖**：游戏盘面自动适配浏览器窗口，以最大正方形比例撑满屏幕（基于 CSS `100vmin` 布局）。
*   **高清像素级渲染**：采用 32px 基础单元重构，所有单位（坦克、砖块、子弹）大而清晰，边缘锐利，适合所有年龄段玩家。
*   **动态气象系统**：Boss 出现时会随机触发 **暴雨、大雪、狂风、雷电** 四种天气特效。

### 2. 合作与竞技
*   **双人实时记分**：顶栏 HUD 实时显示 P1 和 P2 的得分。摧毁普通坦克得 100 分，摧毁 Boss 得 500-1000 分。
*   **共享生命池**：玩家 A 和 B 共用生命总数（❤️）。只要池中有命，阵亡后 2 秒即可在大本营两侧复活。
*   **队友伤害免疫**：玩家之间、敌人之间互不伤血，避免背后挨冷枪。

### 3. 战斗系统增强
*   **爆炸范围伤害 (Splash Damage)**：子弹击中后产生半径爆炸，能一次性炸掉一圈砖块（打出一个“坑”）。
*   **三级进化系统**：
    *   **初始**：普通移动与射击。
    *   **吃 1-2 颗星**：射速倍增，子弹加速并带有发光特效。
    *   **终极形态 (3星)**：子弹变为金黄色，获得**摧毁钢板**的能力！
*   **子弹抵消**：敌我子弹在空中相撞会互相抵消并产生火花，可用于战术防御。

### 4. 搞怪随机事件
*   **多样化 Boss**：随机降临的巨型 Boss（😈、🤡、👹等），体型随机（2-4倍）、血量超厚，击败后会喷发大量稀有道具。

## 🎮 操控说明

| 玩家 | 移动 | 射击 | 颜色 |
| :--- | :--- | :--- | :--- |
| **玩家 1 (P1)** | `W` `A` `S` `D` | `空格键 (Space)` | 黄色坦克 🟡 |
| **玩家 2 (P2)** | `↑` `↓` `←` `→` | `小键盘 Enter` | 蓝色坦克 🔵 |

## 🎁 道具列表
*   ❤️ **爱心**：共享生命 +1。
*   ⭐ **五角星**：坦克等级提升（射速、速度、破坏力）。
*   💣 **炸弹**：全屏敌人瞬间爆炸。
*   🛡️ **护盾**：获得 10 秒无敌时间（坦克周围有白色圆环）。
*   🏗️ **铁铲**：大本营保护墙临时升级为钢板，持续 10 秒。

## 🛠️ 安装与运行
1.  确保您的电脑安装了现代浏览器（Chrome, Edge, Firefox 等）。
2.  将项目中的 `index.html`, `style.css`, `game.js` 放在同一文件夹下。
3.  双击 `index.html` 或将其拖入浏览器即可开始战斗。
